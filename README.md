# Snake
Snake is a game where the player maneuvers the end of a growing line called a snake.
The object is to keep the snake from colliding with iteslf. As the snake eats food the
it will grow in size making the maneuvering more difficult.

## Dependecies
- Pygame
- Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Pygame.
